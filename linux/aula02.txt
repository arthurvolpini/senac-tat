Comandos de manipulação de diretórios
ls = listar arquivos ou diretório 
ls -lh = listar arquivos de forma humana
ls -lha = listar todos arquivos de forma humana incluindo arquivos ocultos
arquivo cor azul: diretório 
arquivo cor branca: arquivo texto
azul claro: Link simbólico (não existe/atalho)
marca texto verde: perigo
arquivo que inicia com ponto (.) quer dizer que é um arquivo oculto
cd / = mudar de diretórios 
cd ~ = diretório pessoal
cd - = voltar uma vex
cd .. = descer um nível
pwd = qual diretótio você está
history = histórico de comandos
! = executar um comando da linha de referência (comandos antigos)
mkdir = comando pra criar diretórios
mkdir -v = verboso (mostra de forma mais detalhado)
'nome' = criar diretório com espaço
ajuda de comandos = --help
mkdir -pv = cria um diretório e infos dentro deste diretório. (mkdir -pv 'nome pasta'/'nome da info' 'nome diretorio/'nome da info') 
rm = remover arquivos e diretório
-f = forçar
-v = verboso
-R = recurssivo
